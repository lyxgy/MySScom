## FAQ：

### 是否需要从头开始编写项目的能力

答：取决于你的目标和任务。如果是修复 Bug 或添加新功能，可能不需要深入理解底层代码；如果是优化性能或全面掌握项目，则需要深入理解。

### 如何高效熟悉项目代码

答：从高层开始，逐步深入：项目功能，结构，核心代码，逐步深入底层代码

使用调试工具：断点，查看变量值，单步调试

编写测试代码：单元测试函数，集成测试，验证交互功能

与团队成员沟通：参与代码审查，请教经验丰富的人

记录学习过程：核心函数逻辑流程图，遇到的问题和解决方案
